Planner — Setup

Connect your Firebase project to enable Google sign-in and real-time sync for your team.

One-time setup

1
Go to console.firebase.google.com → open your coradam-planner project
2
Authentication → Sign-in method → Google → Enable → Save
3
Authentication → Settings → Authorized domains → Add your GitHub Pages domain (e.g. yourname.github.io)
4
Project Settings (⚙) → Your apps → </> Web app → Register app → copy the firebaseConfig object
5
Realtime Database → Rules → deploy the database.rules.json from this repository using firebase deploy --only database. ⛔ Never use the default console rules ({ ".read": true, ".write": true }). These allow the entire internet to read and write your database.
6
Paste your Firebase config JSON (all keys in double quotes) below

Sign in with your Coradam Google account to continue.

Only @coradam.com accounts are authorised.
