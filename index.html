First-time setup: connect to a free Firebase database so all your team's data syncs in real time — no imports needed.

Quick setup (5 minutes, free)

1
Go to console.firebase.google.com → Add project
2
In your project: Build → Realtime Database → Create database — choose your region
3
⚠️ Important — open the Rules tab and replace the contents with:
{ "rules": { ".read": true, ".write": true } } Then click Publish.
4
Back on the Data tab, copy your database URL. It looks like:
https://my-app-default-rtdb.europe-west1.firebasedatabase.app
You can also append a custom path like /coradam-planner to namespace your data.
5
Paste the URL below and click Connect
Already have a database? The most common cause of connection errors is security rules set to "locked mode". Go to Firebase Console → Realtime Database → Rules and make sure .read and .write are both true.

Your data stays in your own Firebase project. Free tier supports teams up to ~20 active users.
